2025/08/24時点の[mu-route-app](https://github.com/rrridaY/mu-route-app)を複製して、写真測量用に改造。

### 使い方
フォトグラメトリのための写真撮影で必要な写真枚数と時間を見積もりできるツールです。
地図内にノードと線を配置し、撮影間隔（メートル）を決めることで撮影に掛かる時間を推定できます。

### 実装したい機能
- 辺の長さがわかる
- 撮影間隔mの入力で所要時間と撮影枚数がわかる

```
# terminal 1
cd backend
npm install
npm start

```

```
# teminal 2
cd app
npm install
npm start

```
